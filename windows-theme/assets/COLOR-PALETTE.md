# GitHub Dark - Complete Color Palette

## Overview

Complete color system for the GitHub Dark Windows theme. All colors are carefully selected for consistency, accessibility, and visual harmony.

---

## 🎨 Base Colors

### Canvas & Backgrounds

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| **Canvas** | `#0d1117` | `13, 17, 23` | Desktop background, deepest layer |
| **Surface** | `#161b22` | `22, 27, 34` | Panels, cards, elevated elements |
| **Elevated** | `#21262d` | `33, 38, 45` | Modals, dropdowns, hover states |

## 🖼️ Asset Color References

For wallpaper and graphic assets, use these gradient presets:

### Background Gradient
```css
background: linear-gradient(180deg, #0d1117 0%, #161b22 100%);
```

### Accent Gradient
```css
background: linear-gradient(135deg, #1f6feb 0%, #a371f7 100%);
```

### Surface Pattern
```css
background: repeating-linear-gradient(
  45deg,
  transparent,
  transparent 10px,
  rgba(48, 54, 61, 0.1) 10px,
  rgba(48, 54, 61, 0.1) 20px
);
```
