# LinkUp Studio - Theme Engine

A comprehensive theme customization system for developers.

## Features

### ◐ Theme Manager
- **Pre-built themes**: GitHub Dark, Dracula, One Dark, Nord, Solarized Dark
- **Create custom themes**
- **Edit existing themes**
- **Apply with one click**

### ◇ Icon Manager
- **Icon packs**: LinkUp Studio, Geometric, Classic, Square, Modern
- **Import custom packs**
- **Preview before applying**

### ▣ Wallpaper Manager
- **Built-in wallpapers**: Gradients
- **Import custom wallpapers**
- **Quick apply**

### ◻ Font Manager
- **Coding fonts**: JetBrains Mono, Fira Code, Consolas, Source Code Pro
- **Preview with sample text**
- **Import custom fonts**

### ◈ Color Manager
- **Accent colors**: Blue, Green, Red, Yellow, Purple, Pink
- **Custom color picker**
- **Color palette generator**
- **Save custom colors**

### ◑ Export/Import
- **Export theme settings**
- **Import theme files**
- **Presets**: Developer, Minimal, Productive

## Usage

### Applying a Theme
1. Go to Themes section
2. Click "Apply" on desired theme
3. Theme is applied immediately

### Creating Custom Theme
1. Click "Create Theme"
2. Customize colors and settings
3. Save and apply

### Exporting/Importing
1. Go to Export/Import section
2. Select components to export
3. Click "Export Theme"
4. Share the .json file
