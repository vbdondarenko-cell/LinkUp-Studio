# LinkUp Studio - Automation Engine

A powerful automation system for developers with scripts, hotkeys, macros, and scheduling.

## Features

### ⚡ Dashboard
- Automation statistics
- Quick actions grid
- Recent activity feed
- Status filters

### ▤ Scripts
- JavaScript/Python/Bash scripts
- Code editor view
- Run/Edit/Delete actions
- Active/Paused status

### ⌨ Hotkeys
- Custom keyboard shortcuts
- Multiple key combinations
- Enable/disable toggles
- Conflict detection

### ◐ Macros
- Multi-step automation
- Sequential actions
- Visual step editor
- Run macros manually

### ◉ Scheduler
- Cron-like scheduling
- Daily/Weekly/Interval
- Run now option
- Pause/resume schedules

### ◑ Logs
- Execution history
- Success/Error/Warning
- Filter by type
- Duration tracking

## Quick Actions

| Action | Description |
|--------|-------------|
| Git Sync | Sync all repositories |
| Build All | Run all build commands |
| Test All | Execute test suites |
| Backup | Backup all projects |
| Deploy | Deploy to production |
| Cleanup | Clean temp files |

## Usage

### Creating a Script
1. Click "New Script"
2. Enter script name
3. Enter script code
4. Click Run to execute

### Creating a Hotkey
1. Click "New Hotkey"
2. Enter key combination
3. Enter action name
4. Press the hotkey to trigger

### Creating a Macro
1. Click "New Macro"
2. Enter macro name
3. Add steps
4. Run or schedule the macro

### Scheduling
1. Click "New Schedule"
2. Set time and repeat
3. Select automation to run
4. Enable/disable as needed
