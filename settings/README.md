# LinkUp Studio - Settings

A comprehensive settings panel for customizing your LinkUp Studio experience.

## Settings Sections

### ◈ General
- **Startup**: Launch on startup, restore session, show splash screen
- **Updates**: Auto-check for updates, update channel (Stable/Beta/Insider)
- **Language & Region**: Language, date format

### ◐ Appearance
- **Theme**: GitHub Dark, Dracula, Nord, Solarized Dark
- **Icons**: LinkUp Studio, Geometric, Classic
- **Fonts**: JetBrains Mono, Fira Code, Consolas, Source Code Pro
- **Layout**: Sidebar position, tab bar position

### ◎ AI
- **OpenHands**: API endpoint, API key, connection status
- **Model Configuration**: AI model, temperature, max tokens
- **AI Behavior**: Auto-save conversations, suggestions, code completion

### ◉ Workspace
- **Projects**: Default folder, recent projects limit
- **Terminal**: Default shell, theme, font size
- **File Explorer**: Show hidden files, confirm before delete

### ▣ Performance
- **Resources**: Memory usage, CPU usage
- **Optimization**: Hardware acceleration, reduced motion, auto-save
- **Caching**: Cache size, file indexing

### ⌨ Shortcuts
- Full keyboard shortcuts list
- Reset to defaults
- Export/Import shortcuts

### ◑ Integrations
- GitHub (Connected)
- GitLab
- Docker
- Slack

### ○ About
- Version information
- Build details
- Links to documentation

## Architecture

settings/

## Features

- Real-time settings preview
- Keyboard shortcut customization

## Usage

Navigate through sections using the sidebar. Changes are saved automatically.
