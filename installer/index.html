Administrator privileges are required for installation. The installer will automatically request elevation when needed.

Requirements:
  • Windows 10+ or Windows 11
  • PowerShell 5.1+
  • Windows Terminal (recommended)
  • VS Code (optional)
Manual Installation:

git clone https://github.com/vbdondarenko-cell/LinkUp-Studio.git
cd LinkUp-Studio\installer
.\OneClick-Install.bat
